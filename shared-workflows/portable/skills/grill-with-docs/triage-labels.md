# Triage label vocabulary

Map the repo's real labels to the five canonical triage roles.

Use this file only if the repo actually uses triage labels or states.

## Canonical roles
- `needs-triage`
- `needs-info`
- `ready-for-agent`
- `ready-for-human`
- `wontfix`
