# GitLab issue tracker

This repository tracks work in GitLab Issues.

Use this file only if the repo actually uses GitLab Issues.

## Notes
- Use the GitLab CLI (`glab`) or the repo's GitLab workflow for issue operations.
- Keep issue titles short and outcome-focused.
- Prefer linking related docs from the issue body when helpful.
- Label names and project conventions are defined in the repo's `docs/agents/` files.
